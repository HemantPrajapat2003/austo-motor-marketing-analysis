# 🚗 Austo Motor Company – Marketing Campaign Analysis

## 📌 Project Overview

Austo Motor Company is a leading automobile manufacturer specializing in **SUVs, Sedans, and Hatchbacks**. The company aims to evaluate the effectiveness of its existing marketing campaign and improve customer targeting strategies using data analytics.

This project performs an end-to-end exploratory data analysis (EDA) on customer purchase data to derive actionable insights that can help optimize marketing decisions and increase sales conversion.

---

## 🎯 Objectives

The primary objective of this analysis is to develop an improved marketing strategy by identifying customer groups with similar purchasing behavior.

Key objectives include:

* Understanding customer demographics and purchasing patterns
* Exploring relationships between customer attributes and vehicle preferences
* Validating business assumptions using statistical and visual analysis
* Identifying potential customer segments for targeted campaigns
* Providing data-driven recommendations to improve marketing efficiency
